# Мова програмування Piton (v1.0) 🐍

**Piton** - це високорівнева скриптова мова з транскрипційним україно-англійським синтаксисом, розроблена для усунення мовного бар'єру за допомогою фонетичної трансляції українських технічних термінів латиницею ("Як чую, так і пишу англійськими літерами").

Проєкт реалізує повноцінний інтерпретатор, написаний на Go (Golang), що складається з лексера, парсера (AST) та обчислювача (evaluator).

## 🌟 Основні філософія та особливості

* **Зрозумілість:** Використання слів замість складних символів для математичних та логічних операцій.
* **Педагогічний підхід до помилок (Instructional Error Handling):** Замість сухого "Syntax Error", інтерпретатор Piton видає "викладацьке" зауваження:
  > `Ryadok [N]: Ya tut interpretator, ya znayu yak maye buty. A tak yak ty pyshesh, tak buty ne maye!`
* **Гнучка структура:** Інтерпретатор завжди сканує код на наявність функцій (`functia`) до початку виконання. Точка входу в програму - виклик головної функції `main()`.

## 🚀 Встановлення та запуск

Вам знадобиться встановлений **Go** (версія 1.20 або вище).

### Запуск скрипта

Щоб швидко запустити програму на Piton:

```bash
go run ./cmd/piton main.piton
```

### Збірка бінарного файлу

Ви можете скомпілювати інтерпретатор у виконуваний файл:

```bash
go build -o piton ./cmd/piton
./piton main.piton
```

## 📖 Синтаксис та ключові слова

### Оголошення та введення/виведення

* **`functia name():`** - Оголошення функції.
* **`vernuty`** - Повернути значення з функції (return).
* **`nekhay a = 3.1`** - Оголошення змінної.
* **`vvid x`** - Введення значення з клавіатури в змінну `x` (призупиняє програму, очікуючи введення користувачем).
* **`drukuvaty`** - Виведення тексту або значення змінної в консоль.

### Математичний апарат

Для підвищення читабельності Piton використовує слова:

* **Корінь:** `korin ( вираз )` або `korin x`
* **Степінь:** `x stupin y`
* **Десятковий логарифм:** `loh10 ( x )` або `loh10 x`
* **Тригонометрія:** `arksyn ( x )` або `kosynus ( x )`
* **Модуль:** `abs x`
* **Базові операції:** `+`, `<`, `>`, `=` (базові символи також підтримуються).

### Керуючі конструкції (Умови)

* **`yaksho`** - `if`
* **`inackshe yaksho`** - `else if` (elif)
* **`inackshe`** - `else`

Блоки коду формуються за допомогою **відступів (індентації)** - аналогічно Python.

## 📝 Приклад програми (`main.piton`)

```piton
functia main():
    drukuvaty "\n--- Завдання ---"
    drukuvaty task()

functia task(): 
    nekhay a = 3.1

    vvid x

    yaksho x > 1
        y = korin ( a + loh10 x )
        vernuty y
    inackshe yaksho abs x < 1
        y = arksyn x
        vernuty y
    inackshe
        y = x stupin a
        vernuty y

main()
```

## 🛠 Архітектура проєкту

Проєкт побудований за класичною схемою розробки інтерпретаторів та розбитий на модулі:

* **`cmd/piton`** - Точка входу CLI.
* **`internal/token`** - Означення лексем та ключових слів.
* **`internal/lexer`** - Перетворення вхідного тексту на послідовність токенів.
* **`internal/ast`** - Побудова абстрактного синтаксичного дерева (Abstract Syntax Tree).
* **`internal/parser`** - Парсинг токенів і побудова AST.
* **`internal/evaluator`** - Виконання (інтерпретація) побудованого AST.
* **`interpreter`** - Високорівневий API (Facade), що об'єднує всі внутрішні компоненти в єдиний інтерфейс.
